Contoh file import untuk aplikasi Labantik Genius

- materials-example.csv
    - Header: title,description,content,mascot_id
    - Buka di Excel/LibreOffice lalu simpan sebagai CSV atau XLSX jika perlu.

- quiz-example.csv
    - Header per baris: quiz_title,quiz_description,time_limit,category,question_text,mascot_id,option_1,option_1_is_correct,option_2,option_2_is_correct,option_3,option_3_is_correct,option_4,option_4_is_correct
    - Gunakan nilai 1/true/yes pada kolom \*\_is_correct untuk menandai jawaban benar.

Catatan:

- Ukuran file maksimal: 10 MB.
- Tipe file yang diterima: .csv, .xlsx, .xls
- Jika ingin file XLSX, buka CSV ini di Excel/LibreOffice dan simpan sebagai XLSX.
